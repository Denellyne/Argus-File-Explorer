## HotKeys
Search:

 	Go back a folder: Escape
  	Go up a folder: F1
 
Close App:

	Ctrl + Alt + Q
	AltGr + Q

## Create files and folders

Create files and folders by clicking on the "New File" icon and "New Folder" icon respectively
